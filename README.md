# modeling-methods

Учебный репозиторий по моделированию.

Внутри лежат независимые работы на Python:

- `pz_1/` — практическое занятие: численное моделирование движения космического аппарата в центральном гравитационном поле Земли.
- `lr1/` — лабораторная работа 1: моделирование случайной величины с заданной плотностью двумя методами.
- `lr2/` — лабораторная работа 2: генерация многомерной нормальной выборки с заданными матожиданием и ковариационной матрицей.

## Требования

- Python 3.10+
- зависимости из `requirements.txt`

Установка из корня репозитория:

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
```

Для Windows:

```bat
python -m venv .venv
.venv\Scripts\activate
python -m pip install -r requirements.txt
```

## Запуск

Команды выполняются из корня репозитория.

### Практическое занятие 1

```bash
python pz_1/interface_pz1.py
```

Открывается Tkinter-интерфейс. На вход подаются:

- начальные координаты `x0`, `y0`, `z0`, м;
- начальные скорости `vx0`, `vy0`, `vz0`, м/с;
- начальное время `t0`, конечное время `tk`, шаг `h`, с;
- метод интегрирования: Эйлера или Рунге-Кутты 4-го порядка.

Результат: графики координат, скоростей и траектории после запуска симуляции.

Консольный вариант также есть в `pz_1/method_pz1.py`:

```bash
python pz_1/method_pz1.py
```

### Лабораторная работа 1

```bash
python lr1/main.py
```

Скрипт:

- генерирует выборки размера `n = 1000` методом обратной функции и методом исключения;
- строит теоретическую плотность;
- строит гистограммы и эмпирические функции распределения;
- считает статистику Колмогорова.

Результат сохраняется в `lr1/graphics/`:

- `density.png`;
- `histogram_inverse.png`;
- `histogram_rejection.png`;
- `cdf_inverse.png`;
- `cdf_rejection.png`.

### Лабораторная работа 2

```bash
cd lr2
python main.py
```

Скрипт работает с вариантом 1:

- размерность `n = 4`;
- матожидание `m_X = [1, 0, 1, 0]`;
- ковариационная матрица `K_X` задана в `lr2/main.py`;
- размер выборки `10000`;
- seed генератора `42`.

Контракт выполнения:

- `K_X` должна быть симметричной и положительно определённой;
- размерность `m_X` должна совпадать с `n`;
- размерность `K_X` должна быть `n x n`;
- сгенерированные значения не должны содержать `inf` или `nan`.

Результат сохраняется в `lr2/results_lr2/`:

- первые 20 векторов `U` и `X`;
- матрица `A` из разложения Холецкого;
- сравнение теоретического и выборочного матожидания;
- теоретическая, выборочная и разностная ковариационные матрицы;
- диаграммы рассеяния для пар компонент `X`.
