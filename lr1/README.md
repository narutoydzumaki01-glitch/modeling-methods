# ЛР1

Тема: моделирование случайной величины с заданной плотностью.

Что делает `main.py`:
- генерирует выборки двумя способами;
- строит теоретическую плотность;
- строит гистограммы и эмпирические функции распределения;
- считает статистику Колмогорова;
- сохраняет графики в `graphics/`.

## Запуск

```bash
cd modeling-methods/lr1
python -m venv venv
venv\Scripts\activate
python -m pip install -r requirements.txt
python main.py
```

## Результат

После запуска в папке `graphics/` появляются PNG-файлы с графиками.
